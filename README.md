# Auto-DXCC

A tool for automatically retrieving [ARRL DXCC Standings](https://www.arrl.org/dxcc-standings) for a list of callsigns.

## Instructions

0. Install **Python 3.13**  
   https://apps.microsoft.com/detail/9pnrbtzxmb4z

1. Run the batch file:  
   `RUN.bat`

2. When prompted, enter the callsign list file name, including the `.xlsx` extension  
   (for example: `callsigns.xlsx`)
